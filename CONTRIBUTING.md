# Contributing to Prompt-XML

Thank you for considering contributing to the Prompt-XML project! We welcome your contributions and suggestions. Here are some guidelines to help you get started:

## XML Prompt Contributions
- When contributing an XML prompt, please ensure that it is well-formed and adheres to XML standards.
- Your XML files should be saved with a `.xml` extension.

## Metadata Requirements
- Each XML prompt should include the following metadata in the root element:
  - **Title**: A brief, descriptive title for the prompt.
  - **Author**: Your name or pseudonym.
  - **Date**: The date the prompt was created or last updated (format: YYYY-MM-DD).
  - **Description**: A brief description of what the prompt is intended for.

Example:
```xml
<prompt>
  <title>Example Prompt</title>
  <author>Your Name</author>
  <date>2026-03-25</date>
  <description>This prompt is an example for illustration.</description>
</prompt>
```

## Quality Standards
- All contributions must be original or properly credited.
- Please test your prompts before submission to ensure they work as intended.
- Follow the project’s code of conduct and be respectful to others.

## Submission Process
1. Fork the repository.
2. Create a new branch for your contribution.
3. Make your changes and commit them with descriptive messages.
4. Push your branch and submit a pull request.

Thank you for helping improve the Prompt-XML project! We appreciate your efforts!
